WHAT WAS ADDED?
───────────────
Image upload functionality for GCash and all payment methods in Admin Dashboard
- Upload QR codes/payment images
- Auto-compress images to save space
- Update or remove images anytime
- Professional, modern UI

WHAT FILES WERE MODIFIED?
─────────────────────────
✓ src/pages/AdminDashboard.jsx (PaymentSettings component enhanced)

NEW FUNCTIONS
─────────────
✓ handleFileUpload(e, methodId) - Upload and compress images
✓ removeQRImage(methodId) - Delete images with confirmation

QUICK STEPS TO USE
──────────────────
1. Admin Dashboard → Payment Methods tab
2. Find GCash payment method
3. Click Edit (pencil icon)
4. Scroll to "QR Code/Payment Image" section
5. Click "Choose Image" or drag file
6. Select PNG/JPG image (any size - auto-compresses)
7. See "Uploading..." feedback
8. Click "Save Changes"
9. ✅ Done! Image visible on GCash card

IMAGE SPECS
───────────
Original Size:  Any (no limit)
Auto-Compressed: 600px width × proportional height
Quality:        0.8 (80%) - optimal for QR codes
Format:         Base64 encoded
Storage:        Supabase payment_settings.qr_url

TWO MODES
─────────

VIEW MODE (showing payment method):
  - Displays QR code (180x180px)
  - "Remove" button below image
  - "Update Image" option to replace

EDIT MODE (editing payment method):
  - Shows current image (120x120px)
  - X button to delete from edit
  - File input to replace
  - "Save Changes" to persist

NEW STATE ADDED
───────────────

  - Disables buttons during upload
  - Shows visual feedback
  - Prevents duplicate submissions

HOW IMAGES ARE STORED
─────────────────────
User selects file
  ↓
Browser converts to base64
  ↓
Client-side compression (600px, 0.8 quality)
  ↓
Upload to Supabase
  ↓
Base64 stored in payment_settings.qr_url
  ↓
UI displays image instantly

TROUBLESHOOTING
───────────────
Upload not working?
  → Check browser console (F12)
  → Verify Supabase connection
  → Try different image file

Image looks pixelated?
  → Normal at 0.8 quality for QR codes
  → Use higher quality source image

Upload slow?
  → Wait for compression to complete
  → Check internet connection

Error message?
  → Check Supabase database status
  → Verify file format (PNG/JPG)
